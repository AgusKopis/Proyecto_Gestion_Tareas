📌 Sistema de Gestión de Tareas

Este proyecto implementa una API REST usando Flask + SQLite con autenticación y cliente de consola.

⚙️ Funcionalidades

  • Registro de usuarios (con contraseñas hasheadas)
  • Inicio de sesión seguro
  • HTML de bienvenida y estructura de tareas
  • Cliente de consola Python
