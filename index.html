🗂️ Sistema de Gestión de Tareas

Este proyecto fue desarrollado para el trabajo práctico PFO 2, e implementa una API REST en Flask con autenticación, manejo de base de datos en SQLite y un cliente en consola para interactuar con los endpoints.

⚙️ Funcionalidades

  • Registro de usuarios con contraseña hasheada
  • Inicio de sesión seguro
  • Creación y visualización de tareas por usuario
  • Cliente de consola en Python
  • Capturas de pruebas en carpeta /capturas

📍 Endpoints principales

  • POST /registro — Registrar usuario
  • POST /login — Iniciar sesión
  • POST /tareas — Crear tarea
  • GET /tareas?usuario=nombre — Ver tareas
